# PokemonGame 🎮qu
jeu de quizz pokémon, petit exercice JS. 
Le but ici est de s'entrainer à utiliser JS pour un petit projet personnel. 


# but du jeu 
Il faut écrire correctement le nom du pokémon affiché pour passé au suivant.
